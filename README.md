## Overall Directory Structure
```

<root-project>/
  |- server/
  |  |- config.php (connect db)
  |  |- <server files> (post, get when you submit form)
  |- vendor/
  |  |- admin/ (css, js, ... of Dashboard)
  |  |- css/ (Your css)
  |  |- img/ (Your img)
  |  |- js/  (Your js)
  |- views/
  |  |- layouts/
  |  |- pages/

```
